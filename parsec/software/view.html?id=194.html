Mercurial Repository
2013-01-01

Development of PaRSEC and DPLASMA is done through a mercurial repository hosted by bitbucket. The public version is updated sporadicaly, mainly around the releases. Collaborators who need access to the development version must have a bitbucket account and be registered with the project to access the mercurial repository. Requests to be added to the DAGuE developers should be sent by mail to George Bosilca.

The main repository can be found at

  http://bitbucket.org/bosilca/dague.public
