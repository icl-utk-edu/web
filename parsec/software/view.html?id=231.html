PaRSEC / DPLASMA 2.0.0-rc1
2015-06-08

The new major version of the PaRSEC runtime is almost ready for primetime. Compared with the previous versions countless new features have been added, the performance has been improved, and support for accelerator reworked. Moreover, tPaRSEC 2.0.0 comes bundled with a new DPLASMA library supporting more P-BLAS functionality with better performance.
